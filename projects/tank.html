Smart Tank System

Status: Work in Progress

About the Project

Smart Tank System is an automation project designed to monitor and control water levels using embedded hardware and smart control logic. The system aims to reduce water wastage and enable efficient monitoring.

Key Features (Planned)

  • Automatic water level monitoring
  • Motor control using relays
  • Low power embedded design
  • Mobile / web control (future scope)

Tech Stack

  • ESP32
  • Relay Module
  • Sensors & Hardware Integration
  • Embedded C / Arduino

Current State

Hardware setup and control logic are under active development. Further improvements will include remote monitoring and automation rules.
