Smart Tank System

Status: Work in Progress

About

The Smart Tank System is an embedded automation project focused on monitoring and controlling water levels efficiently. It is designed to reduce water wastage while enabling smart control of pumps.

Key Features

  • Automatic water level monitoring
  • Motor control using relay modules
  • Sensor-based automation
  • Future scope for remote monitoring

Tech Stack

  • ESP32 Microcontroller
  • Relay Module
  • External 5V power supply
  • Embedded C / Arduino environment

Current State

Hardware integration and control logic are currently under development. Upcoming work focuses on stability, monitoring, and smart automation rules.
